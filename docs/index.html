CloudNC

At CloudNC I was responsible for the UI, layout and styling for internal products, one of which was a complex software application. Sitting behind the frontend was an engine that took a 3D file and generated the most quick and efficient machining toolpaths from all possible permutations. This was then sent to the frontend for rendering in an interactive 3D simulation.

When joining the team I built out the design and implementation of the public, customer facing website, which involved coding web components in Angular and HTML, flavoured with view encapsulated Sass / CSS.

QBE Europe

Working in conjuction with an Umbraco Developer, Technical Director and Account Director, I worked as the sole Frontend Developer on this project. I created reusable HTML/Sass templates and components that could be easily integrated with the CMS, and ensured backwards compatibility for older browsers was implemented to maintain a pleasant experience for the less modern user.

Jack Perfume

Richard E. Grant is not only an actor, he is a man of many talents. Jack Perfume is his fragrance creation, and I build the product website using Shopify, HTML and Sass. The homepage uses looping animations to cycle through boxed and unboxed views of the product, as well as illustrating the product spinning on its axis.

Kallo

I worked with one other developer to build Kallo's revamped website, which sits on a WordPress backend. It uses a multitude of Sass mixins and shared, reusable components which keep things lightweight and fast. This ensured new pages and features could be developed quickly and presented to the client.

Anna Freud National Centre

I helped create the website for children's charity, Anna Freud National Centre. While the version I worked has since been redeveloped, I took advantage of using the BEM (Block, Element Modifier) methodology to implement page templates and accompanying Sass / CSS.

Additional clients include: GSK / KPMG / Land Securities / Microsoft XBOX / O2 / Paymentsense / The Premier League / The Society of Authors / SAP Banking / Shazam / Yeo Valley …and many more.

With an affinity for crafting beautiful user experiences, my skills are deeply founded in elicitation of user requirements, UX, UI design and frontend implementation. Working across these domains, I use Sketch, Figma and Photoshop and program in HTML and Sass with additional JavaScript.

In my most recent role, I achieved success through the design and implementation of three major pieces of work spanning nearly three years, one of which I led and managed ensuring imposed deadlines were adhered to.

Each of these pieces used Angular Material with implementation of custom theming and markdown typography rules to achieve consistency in using common, shared styles.

Usability and intuitive experiences are key to the entire process of a project/product, and so I always build with the end user considerations in mind.
